# Release Notes

## v0.1.0

Initial public release of WARDEN Agent Audit.

- Intent declarations for agent sessions
- Action ledger model with status, reasoning, parameters, and evidence references
- Scope policy checks for action kinds, target classes, denied targets, and session action limits
- Intent drift detector
- Claimed-history detector for prior-work assertions missing from the ledger
- JSON and Markdown report output
- Synthetic fixtures and public-surface scanner
